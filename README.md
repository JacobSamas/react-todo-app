# React To-Do Application

## Overview
This is a simple To-Do application built with React, Redux, and Bootstrap. It allows users to add, view, edit, delete, and filter tasks. The tasks are stored in local storage to persist between page reloads. The application also supports dynamic task lists.

## Features
- Add tasks with due dates and list categories
- Edit and delete tasks
- Mark tasks as complete
- Filter tasks by categories: Upcoming, Today, and View All
- Manage task lists: Personal, Work, and custom lists
- Persistent storage with local storage
